# My Hermes Desktop

A desktop GUI companion for [Hermes Agent](https://github.com/NousResearch/hermes-agent), built with Tauri 2, React 18, and TypeScript.

## Positioning

A native-feeling macOS companion app for Hermes Agent: minimal terminal exposure, low setup friction, and direct access to chat, files, sessions, memory, dashboard tools, snapshots, and an embedded terminal.

## Development

```bash
npm install
npm run tauri dev
```

## Build

```bash
npm run tauri build
```

## Notes

Hermes CLI must be installed separately and available in PATH.
